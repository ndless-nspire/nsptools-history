What for?
--------

PolyDumper lets you dump your TI-Nspire:
- Boot Code 1		( to boot1.img.tns	)
- Boot Code 2		( to boot2.img.tns	)
- Diagnostic Software	( to diags.img.tns	)
- Operating System	( to os.img.tns		)
